- Assembly language programs
- CPU organization and memory concepts

---

- Object-Oriented Programming concepts
- Classes, Inheritance, Polymorphism, Encapsulation
- STL and template programming
- File handling and exception handling

---

- Socket programming in Python/C
- TCP/IP protocol implementation
- Client-server communication programs
- Network topology simulations

### 📖 PL — Programming Languages

- Compiler design basics

### 🐍 Python

- File handling and OS operations
- OOP in Python
- Mini projects and utilities

---

### 🌍 Web Development

This repository contains all practical assignments and mini-projects completed during the **2nd year of B.Tech Computer Engineering** at BVM. Each folder corresponds to a core subject from the curriculum and includes well-commented code, lab programs, and project files.

The work here reflects hands-on learning across fundamental CS topics — from low-level architecture to web development — building the foundation for advanced software engineering.
